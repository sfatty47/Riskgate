# Support

## Where to Ask Questions

- Use **GitHub Discussions** for usage questions and ideas
- Use **GitHub Issues** for confirmed bugs and feature requests

## Before Opening an Issue

- Check existing issues and discussions first
- Include your workflow YAML and `.riskgate.yml` (redact secrets)
- Include logs from the failed GitHub Actions run

## Maintainer Scope

This project is maintained in spare time. Responses may take a few days.
