# Security Policy

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately by emailing: **security@riskgate.dev**

Include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact and affected versions

## Response Expectations

- Initial response target: **72 hours**
- Triage and severity assignment: **7 days**
- Fix timeline depends on severity and complexity

## Supported Versions

RiskGate is currently in early release. Security fixes are provided for the latest release line.
